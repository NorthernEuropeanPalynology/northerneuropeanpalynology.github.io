Object Detection

Object detection is the process of identifying and locating objects within an image or video frame. In contrast, segmentation goes a step further by precisely delineating object boundaries, providing pixel-level accuracy in identifying object shapes within an image or video. Different studies have suggested solutions for the detection or segmentation of pollen grains and non-pollen palynomorphs and we try to implement these algorithms based on the same methodological approach.

Pollen detection based on von Allmen et al. (2024)

In a previous study by von Allmen et al. (2024), we published code that we have since adapted and transferred to TensorFlow.js, a library for machine learning in JavaScript. This code has now been integrated into this HTML webpage, enhancing its accessibility for web-based applications.
Utilizing TensorFlow.js allows the machine learning functionalities to run directly in web browsers, eliminating the need for server-side computations. This integration not only showcases the versatility of the published code but also opens avenues for interactive and user-friendly machine learning applications on the web.
